## 🛣️ RoadMap — Learning Journey Tracker

A modern learning roadmap management web application built using HTML, CSS, and JavaScript that helps users create structured learning paths, track milestones, monitor progress, receive smart recommendations, and stay accountable throughout their learning journey.

## 🚀 Features
* **Multi-Roadmap Management** – Create and manage multiple learning roadmaps.
* **Roadmap Categories** – Organize roadmaps by Development, Security, Data Science, Design, DevOps, and more.
* **Milestone Creation System** – Add milestones with titles, descriptions, and notes.
* **Deadline Tracking** – Set milestone deadlines and monitor due dates.
* **Milestone Editing** – Update milestone details anytime.
* **Milestone Deletion** – Remove unnecessary milestones.
* **Drag & Drop Reordering** – Rearrange milestones dynamically.
* **Progress Tracking** – Track roadmap completion percentage.
* **Milestone Status Management** – Mark milestones as Not Started, In Progress, or Completed.
* **Analytics Dashboard** – Track completed tasks, in-progress tasks, total roadmaps, and weekly progress.
* **Smart Recommendation Engine** – Suggests next learning topics based on completed milestones.

## 🎮 How to Use
**Create a Roadmap**
* Click New Roadmap
* Add title, category, and description
* Create your learning path

**Add Milestones**
* Open roadmap
* Click Add Milestone
* Add milestone details and deadlines

**Track Progress**
* Update milestone status
* Move milestones between: Not Started / In Progress / Completed

**View Analytics**
* Track total completed milestones
* Monitor weekly progress
* Check roadmap statistics

**Get Recommendations**
* Complete milestones
* Receive suggested next topics
